# V2X-InD-conflict-analysis
Vehicle to Everything conflict analysis at the Intersection using InD dataset recorded at the Intersection in Germany.

https://github.com/user-attachments/assets/a108eb6c-2d52-4e32-b81b-e64ee2dd4eee

### InD dataset 
* The dataset is a naturalistic road user trajectories recorded in Aachen, Germany using a drone.
* Pedestrians are extracted from the dataset and time offset is applied to create the desired scenario.
